# [廃止] Mono ランタイム用 "令和" 対応モンキーパッチ [![NuGet Package](https://img.shields.io/nuget/v/Toolbelt.Mono.JapaneseEra.Reiwa.MonkeyPatch.svg)](https://www.nuget.org/packages/Toolbelt.Mono.JapaneseEra.Reiwa.MonkeyPatch/)

## - 告知 -

少なくとも 2020/04/24 現在、Blazor WebAssembly v.3.2 Preview 5 では、和暦年号 "令和" に対応されていました。

よって本プロジェクトは廃止となります。

未だ Blazor WebAssembly v.3.2 Preview 4 またはそれ以前をお使いの場合は、[過去の README](https://github.com/jsakamoto/Toolbelt.Mono.JapaneseEra.Reiwa.MonkeyPatch/blob/master/README-OLD.md) を参照ください。
